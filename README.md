# ⛩ 陰陽五行・神社診断

@3zantn 作・配布の神社診断ツール

---

## 📌 概要

生まれ年から陰陽五行説に基づく属性（木・火・土・金・水）を診断し、
相性の良い神社を全国各地から紹介するブラウザアプリです。
お住まいの都道府県を選ぶと、近い順に神社を表示します。

> ⚠️ **本ツールはエンタメ目的でAIを活用して制作しています。**
> 神社の情報・五行の解釈・距離計算等に誤りが含まれる可能性があります。
> 参拝の際は公式サイト等で必ず最新情報をご確認ください。


---

## 🗂️ 主な機能

**1. ご利益選択**
- 縁結び・学業・商売繁盛・健康・勝運・開運・安産・旅行安全・芸能・農業の10種から複数選択可
- 選ばなくてもOK（全神社が対象になります）

**2. 都道府県選択**
- 47都道府県から選択すると、神社を近い順に表示
- スキップすると全国からランダム表示

**3. 五行診断**
- 生まれ年の西暦から十干を算出し、五行属性を自動判定
- 対応：甲・乙 → 木 ／ 丙・丁 → 火 ／ 戊・己 → 土 ／ 庚・辛 → 金 ／ 壬・癸 → 水

**4. 神社表示**
- 五行・ご利益の条件に合った神社を最大6社表示
- 各神社にGoogle マップリンク・ご利益タグ・五行との相性理由を掲載
